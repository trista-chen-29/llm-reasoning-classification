# LLM Reasoning Data Classification

This project compares multiple machine learning pipelines for text classification using the Nemotron dataset.

## Models
- TF-IDF + Naive Bayes
- TF-IDF + SVM
- TF-IDF + Random Forest

## Results
All models achieved 100% accuracy. Naive Bayes was the fastest.

## How to run
```bash
pip install -r requirements.txt
jupyter notebook
```
